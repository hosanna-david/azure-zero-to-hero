# Azure Administrator Learning Roadmap
This roadmap is designed for becoming a professional Azure Administrator with practical, job-ready skills.

## Phase 1 — Cloud & Azure Fundamentals (2–3 Weeks)

### 1. Cloud Concepts
- IaaS
- PaaS
- SaaS
- Public vs Private vs Hybrid Cloud
- High Availability
- Scalability
- Disaster Recovery
  
### 2. Azure Fundamentals
- Azure Regions
- Availability Zones
- Resource Groups
- Subscriptions
- Azure Portal
- Azure CLI
- Azure PowerShell

### 3. Core Azure Services
- Virtual Machines
- Storage Accounts
- Virtual Networks
- Azure App Services
- Azure SQL Database

### Hands-On Labs
Create:
- Free Azure account
- Resource group
- Windows VM
- Linux VM
- Storage account
- Virtual network
  
### Practice Activities
- Deploy a VM
- Connect through RDP/SSH
- Create snapshots
- Resize VM
- Delete resources safely

## Phase 2 — Azure Administration Core (4–6 Weeks)

### Module 1 — Identity Management
Learn
- Microsoft Entra ID (Azure AD)
- Users and Groups
- RBAC
- Conditional Access
- MFA
- SSO

Hands-On Tasks
- Create users/groups
- Assign roles
- Enable MFA
- Configure password reset
- Configure Conditional Access

### Module 2 — Azure Virtual Machines
Learn
- VM sizes
- Availability sets
- Scale sets
- Managed disks
- VM backups
- Extensions
  
Hands-On Tasks
- Deploy Windows/Linux VM
- Configure NSG
- Attach disks
- Configure auto shutdown
- Install IIS/Apache
- Create custom image
  
### Module 3 — Azure Networking
Learn
- VNets
- Subnets
- NSG
- Route Tables
- DNS
- VPN Gateway
- Load Balancer
- Application Gateway
- ExpressRoute basics

Hands-On Tasks
- Create VNet/Subnets
- Configure NSG rules
- Peer VNets
- Create Load Balancer
- Configure Bastion
- Configure VPN

### Module 4 — Azure Storage
Learn
- Blob Storage
- File Shares
- Queues
- Tables
- Access Tiers
- Lifecycle Management
- Replication Types
  
Hands-On Tasks
- Upload blobs
- Configure lifecycle policy
- Create file shares
- Configure private endpoint
- Generate SAS tokens
  
### Module 5 — Monitoring & Backup
Learn
- Azure Monitor
- Log Analytics
- Alerts
- Recovery Services Vault
- Azure Backup
- Site Recovery
  
Hands-On Tasks
- Configure VM backup
- Create monitoring alerts
- Configure Log Analytics
- Recover deleted VM

## Phase 3 — Advanced Azure Administration (4–5 Weeks)

### Module 1 — Governance
Learn
- Azure Policy
- Management Groups
- Resource Locks
- Tags
- Cost Management

Hands-On Tasks
- Create tagging policy
- Restrict VM creation
- Configure budgets
- Apply locks
  
### Module 2 — Security
Learn
- Microsoft Defender for Cloud
- Key Vault
- Identity Protection
- Security Center
- JIT Access

Hands-On Tasks
- Create Key Vault
- Store secrets
- Enable Defender
- Configure JIT VM Access
  
### Module 3 — Automation
Learn
- Azure Automation
- Runbooks
- ARM Templates
- Bicep
- Terraform basics
- Azure CLI scripting
- PowerShell automation
  
Hands-On Tasks
- Automate VM startup
- Deploy infrastructure using Bicep
- Create PowerShell automation scripts
  
## Phase 4 — DevOps & Hybrid Integration (3–4 Weeks)
Learn
- Git & GitHub
- CI/CD basics
- GitHub Actions
- Azure DevOps basics
- Containers
- Docker basics
- Kubernetes basics
- Hybrid cloud basics

Hands-On Tasks
- Create GitHub repository
- Build CI/CD pipeline
- Deploy app to Azure
- Create Docker container
- Deploy container to Azure
