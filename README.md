# Azure Administrator Learning Roadmap
This roadmap is designed for becoming a professional Azure Administrator with practical, job-ready skills.

## Phase 1 — Cloud & Azure Fundamentals (2–3 Weeks)

### 1. Cloud Concepts
- IaaS
- PaaS
- SaaS
- Public vs Private vs Hybrid Cloud
- High Availability
- Scalability
- Disaster Recovery
  
### 2. Azure Fundamentals
- Azure Regions
- Availability Zones
- Resource Groups
- Subscriptions
- Azure Portal
- Azure CLI
- Azure PowerShell

### 3. Core Azure Services
- Virtual Machines
- Storage Accounts
- Virtual Networks
- Azure App Services
- Azure SQL Database

### Hands-On Labs
Create:
- Free Azure account
- Resource group
- Windows VM
- Linux VM
- Storage account
- Virtual network
  
### Practice Activities
- Deploy a VM
- Connect through RDP/SSH
- Create snapshots
- Resize VM
- Delete resources safely
